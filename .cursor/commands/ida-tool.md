# ida-tool

Use the `ida-sof-buddy` mcp server tools to assist the user with the following query, NOTE:The linux binaries have almost complete debug symbols, the windows binaries do not. If you find an identical function on the windows binary as on the linux binary, after doing using strong logic to assert they are the same function, label/rename the windows function to what the linux's function name is. This helps to make the windows ida file more complete over time.
